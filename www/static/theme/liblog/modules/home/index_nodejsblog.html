nodeJSBlog博客系统

交流学习社区，是一个拥有博客功能，社区讨论，分享，评论等功能的一个学习分享平台，后端完全使用nodeJS，数据库使用mysql，基于nodejs的thinkjs框架搭建的。

一. 简介

nodeJSBlog是一个简单易用的Markdown博客系统，它是基于开源框架thinkJS(使用 ES6/7 特性开发 Node.js 框架)开发的nodejs项目 需要mysql数据库支持，具有管理后台功能，更新博客分为普通文章和markdown文章，markdown文章只需要导入你写好的Markdown文件即可。它摆脱了在线编辑器排版困难，无法实时预览的缺点，一切都交给Markdown来完成，一篇博客就是一个Markdown文件。同时也支持评论，代码高亮，分类，标签云，留言板、友情链接、系统设置等常用功能。nodeJSBlog提供了不同的主题样式，你可以根据自己的喜好配置，如果你想自己制作博客主题，也是非常容易的。nodeJSBlog还支持整站静态网页生成，同时有发布相关的配置，使用nginx做反向代理，动静态资源分离，静态缓存等，使您发布后的博客访问秒开。

二. 功能特点

  1. 文章评论
  2. 代码高亮
  3. 文章内容分页
  4. 支持手机端访问
  5. 问答形式小型社区
  6. 登录/注册/第三方登录/个人中心
  7. 后台权限管理(自定义角色、权限)
  8. 头部和底部链接配置
  9. 顶部滚动通告的配置
  10. 注册邮件通知，登陆邮件通知
  11. 找回密码通过注册邮箱里发密码找回链接

三. nodeJSBlog优势

  1. 使用nodejs编写，对前端开发人员有天然的二次开发优势
  2. 一键导入Markdown文章，摆脱后台编辑排版困难，无法实时预览的缺点
  3. 可自定义URL，支持静态/伪静态访问，良好的SEO
  4. 博客，分类，标签，归档
